DiscAnno Features

  • Temporal structures
  • Discourse mode
  • Visualization by graph and or timeline
  • Customization via scheme
  • Project administration
  • Different user roles: Admin, project manager, annotater

Tips and Hints

  • To deselect click in the area where no text is
  • Use the shortcuts to improve the annotation process
  • To annotate a word or passage start marking in the middle of the word
